# Proyek Analisis Sentimen Aplikasi Livin by Mandiri
This project analyzes user sentiment on the Livin' by Mandiri app based on reviews obtained from the Google Play Store. The goal is to understand user perceptions of the app, identify frequently occurring issues, and provide insights that can be used to improve product quality.

## Dataset
The dataset contains user reviews of Livin' by Mandiri collected from the Google Play Store, including:
1. Review text
2. Rating
3. Date
4. Sentiment information (preprocessing labels/model results)

## Project Objectives
1. Classify user reviews into positive, negative, or neutral.
2. Identify the words/issues that users most frequently complain about or appreciate.
3. Provide easy-to-understand visualizations of sentiment trends.

## Final Results
1. Negative keywords were dominated by complaints about login errors, maintenance, and failed transactions.
2. Positive sentiment mostly related to ease of use and the app's new look.
3. The best model on this project achieved an accuracy of around 90%
